# Slice S29: Analytics Dashboard (Thống kê)

## 1. Mục tiêu
Thắp sáng "Nhà hoang" `/admin/` (trang chủ admin hiện tại chỉ có Text chào mừng) bằng giao diện Dashboard trực quan phản ánh tình hình shop.

## 2. Các bước thực hiện

### Bước 1: Dashboard Controller & Aggregation Queries
- **File:** Mở rộng `dashboard.controller.js` hoặc file quản lý admin hiện hành.
- **Top Metrics:** 
  - `totalRevenue`: sum `Order.total_amount` (Chỉ đơn completed).
  - `totalOrders`: count of `Order`.
  - `pendingOrders`: count of `Order` where `status=pending`.
  - `totalCustomers`: count of `Customer`.
- **Alerts:**
  - `lowStockProducts`: Lấy Product (Tên, Ảnh nhỏ, Kho hiện tại) có `stock_quantity <= min_stock_alert`.
- **Top 5 Sản Phẩm Bán Chạy:**
  - Cách dùng DB (nếu không có chức năng groupBy lằng nhằng hoặc JOIN raw): Query `OrderItem` thuộc các đơn `completed`, dùng Reduce nhồi lại và sort LIMIT 5.
- **Biểu đồ (Chart Data 7 days):**
  - Group doanh thu trong 7 ngày gần nhất gán vào chuỗi Date tĩnh.

### Bước 2: Tích hợp View (`views/admin/dashboard.ejs`)
- Vẽ 4 ô Summary Boxes ở phía trên cùng (Revenue, Orders, Pending, User).
- Vẽ Biểu đồ lớn ở giữa (dùng thư viện `CDN Chart.js`, type `bar`). Data JS array được EJS pass xuống `encodeURIComponent(JSON.stringify(array))`.
- 2 Bảng song song bên dưới: Bảng cảnh báo kho, Bảng SP bán chạy.

## 3. Tech Stack
- View: EJS, AdminLTE (hoặc Boostrap tùy biến), Script CDN JS.
- DB: Prisma Aggregate.

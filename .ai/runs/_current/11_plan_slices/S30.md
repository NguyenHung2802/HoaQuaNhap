# Slice S30: Banner Management (Admin & Slider)

## 1. Mục tiêu
Cho phép Admin tự thay đổi các ảnh Banner trang chủ mà không cần can thiệp vào code, giúp website luôn cập nhật các chương trình khuyến mãi mới nhất.

## 2. Các bước thực hiện

### Bước 1: Routing & Backend (Admin CRUD Banner)
- Tạo module `src/modules/banners/banners.controller.js` và `banners.route.js`.
- Gắn route trong `admin.route.js`: `GET /admin/banners`.
- Thao tác:
    - **Lấy danh sách:** `prisma.banner.findMany({ orderBy: { sort_order: 'asc' } })`.
    - **Thêm/Sửa:** Form tải ảnh lên Cloudinary (sử dụng middleware `upload` đã có), lưu link vào `image_url`. Các trường: `title`, `link_url`, `sort_order`, `is_active`.
    - **Xóa:** Xóa bản ghi và (nên) xóa ảnh trên Cloudinary.

### Bước 2: UI Admin (`views/admin/banners/index.ejs`)
- Lưới danh sách banner (Thumb ảnh nhỏ, Tiêu đề, Link, Active/Deactive).
- Nút Thêm mới/Xóa.
- Toggle switch để bật/tắt hiển thị nhanh (Nếu được - MVP là form edit).

### Bước 3: Tích hợp Slider Public (Trang chủ)
- Sửa `home.controller.js`: Fetch toàn bộ Banner có `is_active: true`, sắp xếp theo `sort_order`.
- Cập nhật `views/public/home/index.ejs` (Khối Hero): Thay thế các ảnh banner cứng bằng vòng lặp EJS từ `banners`.

## 3. Tech Stack
- Prisma `banner` model.
- Cloudinary Upload.
- Swiper.js (Frontend).

# Slice S21 — Refined Filters (Listing Page)
- Goal: Nâng cấp bộ lọc phía Public để đồng bộ với thiết kế (Nguồn gốc, Khoảng giá, Trạng thái).

## Checklist chi tiết
- [ ] Database/Seeding (Sửa/Bổ sung):
    - Thêm metadata `origin_country`, `status` cho Sample Products nếu chưa có.
    - Đảm bảo ít nhất 5-10 sản phẩm có nhiều ảnh mẫu.
- [ ] Listing Controller (Backend):
    - Tham số: `category`, `origin`, `minPrice`, `maxPrice`, `status`, `sort`.
    - Dynamic Prisma where clause.
    - Phân trang (Pagination) đồng bộ với bộ lọc.
- [ ] Listing Sidebar (Frontend - `views/pages/shop.ejs` or similar):
    - Xây dựng UI Filter theo mẫu thiết kế (HTML/CSS).
    - Tích hợp Price Range Slider (HTML input range hoặc library).
    - Badge count cho danh mục.
    - Sự kiện lọc (Query strings update).

## Definition of Done
- Lọc được theo Nguồn gốc (Checkbox/Link).
- Lọc được theo Khoảng giá (Min-Max range).
- Lọc được theo Trạng thái (Mới về, Sẵn có, Giảm giá).
- Kết quả lọc hiển thị đúng danh sách sản phẩm.
- Sidebar mượt mà trên Mobile.

# Slice S27: Admin Order Listing & Filter

## 1. Mục tiêu
Thiết kế trang Quản trị danh sách Đơn hàng, nơi các Admin thấy toàn bộ đơn và dễ dàng tra soát (tìm mã, số điện thoại, trạng thái).

## 2. Các bước thực hiện

### Bước 1: Routing & Backend (Danh sách)
- Tạo module `src/modules/admin/orders/orders.controller.js`.
- Gắn route trong `admin.route.js`: `GET /admin/orders`.
- **Query `prisma.order.findMany`:**
  - `include: { customer: select name/phone }`.
  - Pagination (page, limit=15).
  - Conditions (Where):
    - `status` (pending, processing, shipping, completed, cancelled)
    - `q` (tìm kiếm chuỗi chứa trong `order_code` hoặc `customer_phone`).
  - Sort theo `created_at` DESC (mới nhất lên trên).

### Bước 2: Filter Form
- Sử dụng Bootstrap Filter bar tương tự như trang Shop.
- Lọc Trạng thái (dropdown).
- Thanh tìm kiếm theo keyword (q).

### Bước 3: Giao diện lưới đơn hàng (`views/admin/orders/index.ejs`)
- Cột: Mã Đơn | KH & SĐT | Ngày Đặt | Trạng thái (Badge màu) | Tổng tiền | Menu Action (Chi tiết).
- Nút "Xem chi tiết" dẫn sang `GET /admin/orders/:id`.

## 3. Tech Stack
- Express EJS, Prisma FindMany. Pagination layout.

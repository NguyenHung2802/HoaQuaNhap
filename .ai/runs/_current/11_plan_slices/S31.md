# Slice S31: Blog / News (Admin & Public)

## 1. Mục tiêu
Xây dựng một nền tảng tin tức bài viết tại `/blogs` giúp SEO trang web và cung cấp giá trị kiến thức về hoa quả cho khách hàng.

## 2. Các bước thực hiện

### Bước 1: Routing & Backend (Admin CRUD Blog)
- **Route:** `GET /admin/blogs`, `GET /admin/blogs/create`, `POST /admin/blogs`, `GET /admin/blogs/edit/:id`, `POST /admin/blogs/delete/:id`.
- **Thao tác:**
    - Ghi lại `author_id` từ `req.session.user.id`.
    - `slugify` tiêu đề nếu chưa có.
    - `summary`: Đoạn tóm tắt hiển thị ở danh sách.
    - `content`: Lưu HTML soạn thảo từ Rich Text Editor.

### Bước 2: Tích hợp Quill/TinyMCE (Admin)
- Thêm thư viện vào trang Edit/Create của Blog (`<script src="...CDN"></script>`).
- Cấu hình thẻ textarea thành khung soạn thảo hình ảnh/văn bản.

### Bước 3: Public Blog Pages
- **Route:** `GET /blogs` (Danh sách), `GET /blog/:slug` (Chi tiết).
- **UI Public:** 
    - Trang danh sách: Layout Grid 3 cột bài viết (Ảnh | Ngày | Tiêu đề | Tóm tắt | Xem thêm).
    - Trang chi tiết: Bài viết dạng tin tức, có sidebar "Bài viết mới nhất".

### Bước 4: Tích hợp Trang chủ
- Sửa `home.controller.js`: Lấy 3 bài blog mới nhất.
- Cập nhật `views/public/home/index.ejs`: Hiển thị khối tin tức cuối trang chủ.

## 3. Tech Stack
- Prisma `blogPost` model.
- Quill JS (Editor).
- Bootstrap Grid.

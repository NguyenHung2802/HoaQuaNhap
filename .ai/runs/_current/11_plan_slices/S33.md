# Slice S33: Trang thông tin tĩnh (Static Content & FAQ)

## 1. Mục tiêu
Cung cấp các thông tin thiết yếu giúp khách hàng an tâm mua hàng (Chính sách, Giới thiệu, FAQ), hoàn thiện sự chuyên nghiệp của thương hiệu.

## 2. Các bước thực hiện

### Bước 1: Routing & Backend (Admin CRUD Settings)
- **File:** `src/modules/settings/settings.controller.js` và `settings.route.js`.
- **Gắn route trong `admin.route.js`:** `GET /admin/settings/pages`.
- **Thao tác:**
    - Lấy các bản ghi Setting có `group_key="pages"`.
    - Key ví dụ: `about-us`, `delivery-policy`, `refund-policy`, `faq`.
    - `value`: Lưu trữ text/HTML của trang.
    - Cập nhật Setting theo Key.

### Bước 2: UI Public (Trang thông tin)
- **Route Public:** `GET /pages/:key`. (Ví dụ: `/pages/about-us`).
- **UI Public:** 
    - Trang đơn giản: Tiêu đề và Khối văn bản nội dung.
    - FAQ: Tích hợp hiệu ứng Accordion (Sổ xuống/Đóng lại) để gọn gàng.

### Bước 3: Tích hợp Footer
- Cập nhật `views/partials/footer.ejs`.
- Thay thế các liên kết href bằng đường dẫn động phù hợp tới các trang nội dung tĩnh.

## 3. Tech Stack
- Prisma `setting` model.
- Bootstrap Accordion (FAQ).

# Slice 2: Giao diện Hồ sơ & Tải ảnh đại diện (UI Tier)

## 1. Thành phần giao diện (Views)
Cần tạo `src/views/public/profile/index.ejs` với các phần:
- Sidebar trái cho Account (Hồ sơ, Đơn hàng, Đăng xuất).
- Content chính là Tab pane cho:
    1. Thông tin cá nhân (Form có Avatar upload).
    2. Danh sách địa chỉ nhận hàng.

## 2. Thao tác thực thi
1. Tạo thư mục `src/views/public/profile`.
2. Tạo tệp `index.ejs`.
3. Sử dụng Bootstrap Tabs cho UI mượt mà.
4. CSS tùy chỉnh cho Avatar circle (Preview ảnh trước khi lưu).
- Yêu cầu Avatar: Bo tròn, có nút Edit đè lên ảnh.

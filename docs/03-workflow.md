# Quy trình nghiệp vụ

## 1. Quy trình bán hàng

1. Admin thêm sản phẩm
2. Sản phẩm hiển thị ngoài website
3. Khách truy cập website
4. Khách xem sản phẩm
5. Khách thêm giỏ
6. Khách nhập thông tin đặt hàng
7. Hệ thống tạo đơn
8. Hệ thống trừ tồn kho tạm thời hoặc trừ khi xác nhận đơn
9. Admin xác nhận đơn
10. Đơn chuyển sang đang xử lý
11. Đơn chuyển sang đang giao
12. Đơn hoàn thành hoặc hủy

## 2. Quy trình quản lý tồn kho

1. Admin nhập số lượng hàng ban đầu
2. Khi phát sinh đơn hợp lệ thì trừ tồn
3. Khi hủy đơn thì hoàn tồn nếu cần
4. Khi nhập thêm hàng thì cộng tồn
5. Hệ thống cảnh báo khi tồn thấp

## 3. Quy trình quản lý sản phẩm

1. Tạo danh mục
2. Tạo sản phẩm
3. Upload ảnh
4. Gán danh mục
5. Cài giá / khối lượng / xuất xứ
6. Publish hoặc ẩn
7. Theo dõi hiệu quả bán

## 4. Quy trình xử lý nội dung quảng bá

1. Tạo banner
2. Chọn sản phẩm nổi bật
3. Tạo combo / bài viết
4. Đẩy nội dung lên trang chủ
5. Theo dõi hiệu quả chuyển đổi
